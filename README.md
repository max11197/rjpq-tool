# 楓之谷羅密歐與茱麗葉組隊任務 - 研究室平台小工具 (RJPQ-Tool)

這是一款專為《楓之谷》羅密歐與茱麗葉組隊任務 (RJPQ) 內「研究室平台」設計的輔助工具。幫助玩家快速記錄並同步各玩家的正確位置，減少溝通過程中的混亂。

## 🌟 主要功能

- **即時多人同步**：基於 PeerJS (P2P) 技術，無需中轉伺服器，房主建立房間後分享房間代碼即可與隊友協作進行。
- **AI 自動辨識**：支援擷取遊戲視窗畫面，透過 OCR 技術自動偵測對話框內的數字 (如 11111 11111)，並自動在地圖上標記位置。
- **懸浮小視窗 (PiP)**：支援瀏覽器畫中畫 (Picture-in-Picture) 功能，可將平台地圖縮小並置頂於螢幕畫面。
- **長按快速標記**：在平台上長按 150ms 即可彈出 O/X 選項，支援拖曳選擇，操作流暢。
- **無障礙自訂顏色**：支援自訂各玩家的代表顏色，並內建常見色盲模板 (紅色盲、綠色盲、藍黃色盲)。
- **快捷鍵支援**：
  - `A/S/D/F`：快速選取玩家 1~4。
  - `數字鍵 0-9` + `1-4`：快速點亮特定層數與平台 (0 為第 10 層)。
  - `X`：切換當前選取玩家的 X標記模式。
  - `P`：暫停/恢復自動偵測。
  - `R`：重設所有標記。
  - `C`：取消快捷鍵輸入緩衝。

## 🚀 快速上手

1. **建立/加入房間**：開啟 `index.html`，輸入房間代碼並點選「建立」或「加入」。
2. **選取玩家**：點擊下方的玩家按鈕或使用 `ASDF` 鍵選取您目前的身份。
3. **開始標記**：
   - **手動**：直接點擊平台格子標記正確位置。
   - **自動**：點選「自動偵測」並選取正確的遊戲解析度/區域，工具將自動填入隊友報出的數字。
4. **輔助檢視**：點選「懸浮視窗」開啟小窗，方便在遊戲中查看。

## 🛠️ 技術棧

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **P2P Communication**: [PeerJS](https://peerjs.com/)
- **OCR Engine**: [Tesseract.js](https://tesseract.projectnaptha.com/)
- **Visuals**: CSS 玻璃擬態 (Glassmorphism) 設計風格

## ⚠️ 注意事項

- 本工具完全在瀏覽器端運作，不儲存任何個人資料。
- 自動辨識功能受遊戲字體顏色、背景干擾影響，若偵測不準建議改用「手動選取」ROI 區域。
